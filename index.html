Extensões → Apps Script. Apague tudo, cole o código abaixo e altere no topo: EMAIL_ADMIN (seu e-mail) e API_KEY (chave Anthropic, só para leitura de PDF).

      
3
Ative o e-mail diário
Ícone de relógio ⏰ (Gatilhos) → Adicionar gatilho → Função verificarVencimentos · Temporizador por dia · 8h–9h.

      
4
Publique como app web
Implantar → Nova implantação → Aplicativo da Web. Executar como: você. Acesso: Qualquer pessoa. A URL já está travada no sistema — se você republicar com a mesma URL, nada muda. Se gerar uma URL nova, me avise para atualizar.

      
5
Pasta de PDFs (automática)
Na primeira vez que você anexar um PDF, o script cria sozinho a pasta "Contratos FHSL - PDFs" no seu Google Drive e guarda os arquivos lá. O botão "Ver PDF" abre o original.
